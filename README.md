# Comment-Toxicity-Detection
PyTorch LSTM based Comment Toxicity Detection using NLP and Streamlit with real-time and bulk CSV prediction.

This project detects toxic comments using NLP and Deep Learning.

     Technologies Used
- Python
- PyTorch
- Streamlit
- NLP
- LSTM

      Features
- Real-time toxicity prediction
- Bulk CSV prediction
- Streamlit web application

       How To Run

pip install -r requirements.txt

streamlit run app.py
